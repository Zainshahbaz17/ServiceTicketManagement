# Service Ticket Management System

A C# console application that simulates an internal service desk system
used to log, manage, and resolve client support tickets.

## Features
- Create service tickets
- View all tickets
- Update ticket status
- Assign tickets to engineers
- Simple in-memory data storage

## Tech Stack
- C#
- .NET
- Console Application

## Business Context
This project reflects the type of internal tooling commonly used in
managed service and IT support environments.
